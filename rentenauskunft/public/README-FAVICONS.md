# Favicon-Dateien erstellen

## Benötigte Dateien:

Du musst folgende PNG-Dateien aus deinem Logo erstellen:

### 1. **favicon.ico** (16x16, 32x32, 48x48)
- Multi-Size ICO-Datei
- Tool: https://www.favicon-generator.org/

### 2. **favicon-16x16.png**
- Größe: 16x16 Pixel
- Format: PNG

### 3. **favicon-32x32.png**
- Größe: 32x32 Pixel
- Format: PNG

### 4. **apple-touch-icon.png**
- Größe: 180x180 Pixel
- Format: PNG
- Für iOS Home Screen

### 5. **android-chrome-192x192.png**
- Größe: 192x192 Pixel
- Format: PNG
- Für Android

### 6. **android-chrome-512x512.png**
- Größe: 512x512 Pixel
- Format: PNG
- Für Android (hochauflösend)

### 7. **mstile-150x150.png**
- Größe: 150x150 Pixel
- Format: PNG
- Für Windows Tiles

### 8. **og-image.png**
- Größe: 1200x630 Pixel
- Format: PNG
- Für Social Media (Facebook, Twitter, LinkedIn)

## Empfohlene Tools:

1. **Favicon Generator**: https://realfavicongenerator.net/
   - Lädt ein Bild hoch (mindestens 512x512)
   - Generiert ALLE benötigten Dateien automatisch
   - Gibt dir auch den korrekten HTML-Code

2. **Manual erstellen**:
   - Photoshop / GIMP
   - Exportiere dein Logo in verschiedenen Größen
   - Achte auf transparenten Hintergrund (außer og-image.png)

## Farben:

- **Theme Color**: #083163 (Renten-Blau)
- **Background**: #ffffff (Weiß)

## Nach dem Erstellen:

Ersetze die Platzhalter-SVG-Dateien in `/public` mit deinen PNG-Dateien!
